Пошаговая инструкция пользователя:

1) Запустите приложение.

2) Чтобы открыть файл с данными введите полный путь до файла в текстовое поле вверху приложения 
 и нажмите «Открыть файл» или просто нажмите «Открыть файл» для запуска файлового диалога, 
 после чего выберите файл из каталога файлов.

3) Внимательно осмотрите открытую таблицу. 
Проверьте правильность заголовков столбцов и содержимого таблицы.

4) Если в таблице не задана цель, т.е. состояние оценок учащегося, 
 которое мы хотим получить в идеале, поставьте галочку внизу слева от пункта «Ввести цель вручную», 
 чтобы ввести цель.

5) Поставьте галочку слева от пункта «Учитывать неуспевающих студентов»,
 если хотите, чтобы учащиеся получившие неудовлетворительные оценки, тоже участвовали в рейтинге.

6) Нажмите на кнопку «Рейтинг», чтобы приступить к обработке данных. 

7) Если вы отметили пункт «Ввести цель вручную», появится форма для заполнения цели. 
 Впишите соответствующие значения в текстовые поля и нажмите «ОК».

8) Вы можете осмотреть полученную таблицу.

9) Для сохранения результатов нажмите «Сохранить как». Будет запущен файловый диалог.

10) Введите имя под которым хотите сохранить файл и нажмите «Сохранить».

11) Закройте приложение, чтобы закончить работу или выберите новый исходный файл 
 для продолжения работы.
